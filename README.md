# 🌍 HennyClever - Location Spoofer

Chrome extension to spoof your geolocation on Google and other websites.

## Features

✨ **Location Spoofing** - Spoof your location to any world capital
🌐 **Multi-language Support** - Choose language for each country
⭐ **Favorites & History** - Quick access to recent and favorite locations
⌨️ **Keyboard Navigation** - Use arrow keys to navigate dropdown
🔄 **Toggle On/Off** - Enable/disable spoofing without losing settings
🎯 **Whitelist Mode** - Choose specific domains to spoof
🎨 **Beautiful UI** - Modern, gradient design with smooth animations
🚀 **Badge Indicator** - See current country flag on extension icon

## Installation

1. Clone or download this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" (top right)
4. Click "Load unpacked"
5. Select the HennyClever folder

## Usage

1. Click the extension icon
2. Search for a country (type to filter)
3. Select language (optional)
4. Click "Save Location"
5. Visit google.com or any whitelisted site

## Keyboard Shortcuts

- `↑/↓` - Navigate dropdown
- `Enter` - Select country
- `Esc` - Close dropdown

## Settings

**Enable/Disable Spoofing** - Toggle without losing your location
**Whitelist Mode** - Add domains where spoofing should work
**Favorites** - Star countries for quick access

## How It Works

The extension overrides the browser's geolocation API to return fake coordinates. It only works on:
- Google.com (by default)
- Domains you add to whitelist
