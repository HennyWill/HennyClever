# Changelog

All notable changes to HennyClever will be documented in this file.

## [1.0.0] - 2025-12-26

### 🎉 Initial Release

#### Features
- 🌍 Spoof geolocation to 200+ world capitals
- 🔍 Smart search with autocomplete and filtering
- 🌐 Multi-language support (80+ languages)
- ⭐ Favorites system for quick access
- 📜 Location history (last 5 used)
- ⌨️ Keyboard navigation (arrow keys, Enter, Esc)
- 🔄 Quick enable/disable toggle
- 🎯 Whitelist mode for domain-specific spoofing
- 🚀 Country flag badge indicator
- 🎨 Beautiful gradient UI with animations
- 💾 All data stored locally

#### Technical
- Built with Manifest V3
- Service worker architecture
- CSP-compliant injection method
- CustomEvent communication
- Chrome local storage API

#### Privacy
- No data collection
- No external servers
- No tracking or analytics
- Open source code
- Transparent permissions
