# Privacy Policy for HennyClever - Location Spoofer

**Last Updated: December 2025**

## Overview

HennyClever is committed to protecting your privacy. This extension does not collect, store, transmit, or share any personal data.

## Data Collection

**We do NOT collect any data.** Specifically:

- ❌ No personal information
- ❌ No browsing history
- ❌ No location data
- ❌ No analytics or tracking
- ❌ No cookies
- ❌ No external servers or APIs

## Data Storage

All settings and preferences are stored **locally on your device** using Chrome's local storage API:

- Selected country/location
- Favorite locations
- Location history (last 5)
- Whitelist domains
- Extension preferences

**This data never leaves your device** and is only accessible by the extension itself.

## Permissions Explained

The extension requires the following permissions:

### Storage
- **Purpose:** Save your location preferences, favorites, and settings locally on your device
- **Data:** Only settings you explicitly configure

### Scripting
- **Purpose:** Inject geolocation override code into web pages
- **Scope:** Only on domains you whitelist (default: google.com)

### Tabs
- **Purpose:** Detect when whitelisted pages load to apply location spoofing
- **Access:** URL matching only, no content reading

### Host Permissions (google.com)
- **Purpose:** Override geolocation on Google by default
- **Can be extended:** You can add more domains via whitelist

## Third-Party Services

This extension does **NOT** use any third-party services, analytics, or external APIs.

## Data Deletion

To delete all extension data:
1. Right-click the extension icon
2. Select "Remove from Chrome"
3. Or clear Chrome extension data in Settings → Privacy → Clear browsing data → Hosted app data

## Children's Privacy

This extension does not knowingly collect any information from children under 13.
